# Florify — Feature Brainstorm

> สรุปไอเดียฟีเจอร์ใหม่ที่น่าเพิ่ม โดยเรียงตาม impact และความเข้ากับแก่นของเกม
> (Meditative, Discovery, Collection, Daily Ritual, Poetic Worldbuilding)

---

## Tier 1 — High Impact / ตอบโจทย์ Core Loop

### 1. Terrarium (ตู้จัดสวน)
**Pillar:** Collection + Creative Expression

เลือก flora จาก collection มาจัดวางใน terrarium เป็น diorama 3D ส่วนตัว
- วาง flora ได้ 5–9 ตัวบน grid ขนาดเล็ก
- Flora ที่ merge level สูงจะดูโดดเด่นกว่า (glow, size bonus)
- เปลี่ยน background theme ได้ (forest floor, mossy rock, desert sand)
- แชร์เป็นรูปได้เหมือน Florist Card
- **ทำไมสำคัญ:** ให้เหตุผลในการ merge + creative outlet + sharable content

### 2. Season & Event System (ฤดูกาล)
**Pillar:** Discovery + Daily Ritual

ระบบฤดูกาลหมุนเวียน (เช่น ทุก 2 สัปดาห์) แต่ละฤดูมี:
- Limited-edition species 3–5 ตัวที่ดรอปเฉพาะช่วงนั้น
- Theme สี background + banner ที่เปลี่ยนไป
- Season-specific daily missions
- Season pass ง่ายๆ (free track) — ทำ mission สะสมแต้มแลก species/cosmetics
- **ทำไมสำคัญ:** FOMO ที่พอดี + ให้เหตุผลกลับมาเล่นทุกวัน + เนื้อหาหมุนเวียน

### 3. Flora Companion (สัตว์น้อยประจำสวน)
**Pillar:** Discovery + Collection

สัตว์เล็กๆ (ผีเสื้อ, นก, ด้วง, หิ่งห้อย) ที่ปรากฏรอบ mature trees:
- ได้แบบ random เมื่อ harvest ต้นไม้ (10% chance)
- สะสมใน Companion Codex แยกต่างหาก
- วาง companion ใน Terrarium ได้
- Rare companions มาเฉพาะบาง collection / rarity
- **ทำไมสำคัญ:** เพิ่มชั้นของการสะสมโดยไม่รบกวน core loop

## Tier 2 — Medium Impact / เสริม Engagement

### 4. Garden Journal (สมุดบันทึกสวน)
**Pillar:** Poetic Worldbuilding + Daily Ritual

ไดอารี่อัตโนมัติที่บันทึก milestones ของผู้เล่น:
- "Day 1 — You planted your first seed..."
- "Day 14 — A Legendary species appeared for the first time!"
- "Day 30 — Your garden now holds 100 species."
- เขียนเป็น prose สั้นๆ สไตล์ poetic (TH/EN)
- ผู้เล่นเปิดอ่านย้อนหลังได้เหมือนอ่านนิยาย
- **ทำไมสำคัญ:** emotional attachment + unique to each player

### 5. Constellation Map (แผนที่ดาว)
**Pillar:** Collection + Discovery

Visual progression map ที่แสดง collection เป็น "กลุ่มดาว":
- แต่ละ species คือดาวดวงหนึ่ง
- เมื่อ collect ได้ ดาวจะสว่างขึ้น
- เส้นเชื่อมระหว่าง species ใน collection เดียวกัน
- สว่างครบทั้งกลุ่มจะเห็น constellation shape
- **ทำไมสำคัญ:** visual progress ที่สวยงามกว่า list view

### 6. Weekly Challenge (ชาเลนจ์ประจำสัปดาห์)
**Pillar:** Daily Ritual

ภารกิจพิเศษรายสัปดาห์ (reset ทุกวันจันทร์):
- เป้าหมายใหญ่กว่า daily missions (เช่น "Harvest 5 Rare species this week")
- Reward เป็น exclusive cosmetic หรือ sprouts จำนวนมาก
- ทำครบทุกสัปดาห์ในเดือน = monthly reward
- **ทำไมสำคัญ:** medium-term goal ที่ขาดอยู่ระหว่าง daily missions กับ achievements

### 7. Wandering Merchant (พ่อค้าเร่)
**Pillar:** Collection + Strategy

NPC ที่มาเยี่ยมทุก 3 วัน พร้อม rotating inventory:
- ขาย species เฉพาะ 3 ตัว (random, อาจมี rare/legendary)
- ซื้อด้วย Dried Leaves (ใบแห้ง) แทน Sprouts
- ลดดวง luck factor + ให้คุณค่ากับ Dried Leaves มากขึ้น
- **ทำไมสำคัญ:** Dried Leaves มีทางใช้เพิ่ม + safety net สำหรับ completionists

## Tier 3 — Polish & Delight

### 8. Zen Mode (โหมดสมาธิ)
**Pillar:** Meditative

กดปุ่ม "Zen" บนหน้า home เพื่อ:
- ซ่อน UI ทั้งหมด เหลือแค่ต้นไม้ 3D
- ต้นไม้หมุนช้าๆ อัตโนมัติ
- Ambient sounds (ลมพัด, นกร้อง, น้ำไหล)
- ใช้เป็น screensaver / meditation companion
- **ทำไมสำคัญ:** เสริม identity ของเกมว่า "chill & meditative"

### 9. Gardener's Almanac (สถิติสวยๆ)
**Pillar:** Collection

หน้า stats ที่แสดงข้อมูลผู้เล่นแบบ visual:
- Most harvested species (top 5 bar chart)
- Rarity breakdown (pie chart)
- Activity heatmap (เหมือน GitHub contribution graph)
- Collection milestones timeline
- "Your rarest find" spotlight
- **ทำไมสำคัญ:** ข้อมูล stats มีอยู่แล้วใน PlayerState แค่ยังไม่มี UI ที่สวย

### 10. Photo Mode (โหมดถ่ายรูป)
**Pillar:** Creative Expression

กล้องถ่ายรูป 3D tree อย่างจริงจัง:
- เลือก background (gradient, bokeh, nature scene)
- เลือก filter (warm, cool, vintage, noir)
- เลือก frame (rarity-themed, seasonal, minimal)
- Export เป็นรูป share ได้
- **ทำไมสำคัญ:** user-generated content สำหรับ social media

### 11. Passport Background & Theme
**Pillar:** Creative Expression

ให้ customize Florist Card มากขึ้น:
- เลือก background (unlock ตาม achievement/collection)
- เลือก frame style
- เลือก font style
- **ทำไมสำคัญ:** ต่อยอดจาก passport ที่มีอยู่แล้ว ใช้เป็น reward sink

### 12. Flora Stories (นิทานพฤกษา)
**Pillar:** Poetic Worldbuilding

Lore ที่ลึกขึ้นสำหรับ species ที่ merge ถึง Lv 3+:
- ปลดล็อค "Chapter 2" ของ lore ที่ Lv 3
- ปลดล็อค "Chapter 3" ที่ Lv 5
- เรื่องราวต่อเนื่องของต้นไม้แต่ละต้น
- **ทำไมสำคัญ:** ให้เหตุผลในการ merge + ตอบแก่น worldbuilding

## Tier 4 — Future / ต้องคิดเพิ่ม

### 13. Cloud Sync + Friend Garden Visit
เยี่ยมสวนเพื่อน (read-only) — ดู collection, terrarium, passport
ไม่มีการแข่งขัน แค่ชื่นชมกัน ให้ feeling ของ community

### 14. New Rarity: Eternal (นิรันดร์)
Rarity ใหม่เหนือ Legendary — drop rate 0.5%
Species design ที่พิเศษมาก (cosmic/ethereal theme)
ได้มาจาก special event หรือ ultra-rare drop เท่านั้น

### 15. Fungi Kingdom Collection
Collection ใหม่เป็นเห็ดแทนต้นไม้ — visual ที่ต่างมาก
เพิ่มความหลากหลายใน collection

### 16. Mini-Garden Widget
สำหรับ PWA ที่ support widgets — แสดง active tree + water drops count
กดรดน้ำจาก widget ได้เลย
